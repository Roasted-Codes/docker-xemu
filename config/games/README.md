# Game ISOs Directory

Place your Halo 2 game ISOs here. They will be accessible to xemu when the container runs.

**Note:** Game ISO files are not included in this repository due to their large size and copyright considerations.

## Required Files
- **Halo 2 R1E** (xiso format) - The main Halo 2 game disc
- **Halo 2 DLC** (xiso format, optional) - Map pack DLC content

## Format
- `.xiso` - Xbox ISO format (only format supported)

Ensure you own legal copies of Halo 2 before using with this emulator.
